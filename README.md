FOOD-HUB


### Technologies Used

- React.js
- MongoDB
- Node.js
- Express.js
- Json Web Token



## Installation

- Clone the repository
- Go to the project directory and run the following commands to install the dependencies (frontend & backend)

```
npm install
cd frontend
npm install
cd ../backend
npm install
```
